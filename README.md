# 文件分类整理工具

一个实用、好看且高效的文件自动分类整理工具，帮助我彻底解决了文件混乱的问题。

![工具界面截图](screenshot.png)

## 项目成果

我用这个工具成功整理了 **31455 个文件**：

- 成功处理：**29566 个**
- 检测到重复文件：**1888 个**

整理之后，过去混乱散落的照片、文档、视频等文件，现在都按类型 + 年份清晰地归档，找文件变得非常轻松！

## 主要功能

- **智能图片分类**  
  - 有拍摄时间的照片 → `images/Photos/年份/`  
  - 没有拍摄时间的图片（截图、壁纸等） → `images/Images/年份/`

- **文档自动细分**  
  PDF、Word、Excel、Text，并按年份创建子文件夹

- **其他分类**  
  视频、音频、压缩包分别存放

- **重复文件智能处理**  
  自动检测并统一移动到 `Duplicates` 文件夹

- **两种操作模式**  
  - **拷贝模式**（安全，默认推荐）  
  - **移动模式**（彻底整理，源文件夹会变干净）

- 实时进度显示 + 生成详细日志 `duplicates_log.txt`

## 使用方法

1. 运行程序
2. 选择要整理的**源目录**
3. 选择整理后的**目标根目录**
4. 选择操作模式（拷贝 或 移动）
5. 点击「开始整理文件」

整理完成后，目标目录结构示例：

目标目录/
├── images/
│   ├── Photos/2025/
│   ├── Photos/2024/
│   └── Images/2025/
├── documents/
│   ├── PDF/2025/
│   ├── Word/2024/
│   └── Excel/2025/
├── videos/2025/
├── audios/
├── archives/
├── other/
└── Duplicates/

- Tkinter（图形界面）

- pathlib、shutil 等标准库
