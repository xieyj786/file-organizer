# 文件分类整理工具

一个实用、高效的办公文档及网页文件自动分类整理工具。

![工具界面截图](screenshot.png)

## 项目成果

我使用此工具成功整理了 **31455 个文件**：

- 成功处理：**29566 个**
- 检测到重复文件：**1888 个**

现在所有文档、网页文件都按年份和类型清晰归档，查找和管理变得非常方便。

## 主要功能

- **智能文档分类**（按修改时间年份）
  - `.doc`、`.docx`、`.pdf`、`.txt` → `YYYY年/文本文件/`
  - `.mht`、`.htm` → `YYYY年/网页文件/`
  - `.xls`、`.xlsx`、`.ppt`、`.pptx`、`.zip`、`.rar` 等 → `YYYY年/其他文件/`

- **自定义扩展**  
  支持在界面中添加更多后缀到“其他文件”分类。

- **重复文件检测**  
  对 `.doc/.docx/.pdf/.txt/.xls/.xlsx/.ppt/.pptx/.mht/.htm` 等文件进行 **SHA-256 哈希查重**，重复文件自动移动到 `duplicate` 文件夹。

- **两种操作模式**
  - **拷贝模式**（安全，默认推荐）
  - **移动模式**（彻底整理）

- **支持多次汇总整理**  
  可从多个不同源目录，陆续整理到同一个目标目录。

- **自动生成日志**  
  每次运行生成带序号的日志文件（`file_organizer_log.txt`、`file_organizer_log (1).txt` 等），记录处理统计和重复文件列表。

## 使用方法

1. 运行 `file_organizer.py`
2. 选择**源目录**（要整理的文件所在位置）
3. 选择**目标根目录**（可多次使用同一个目录进行汇总）
4. 选择操作模式（拷贝 或 移动）
5. 可选：在“其他文件额外后缀”中添加需要归类的后缀（如 `.7z,.md`）
6. 点击「开始整理文件」

### 整理后目录结构示例

目标根目录/
├── 2025年/
│   ├── 文本文件/
│   ├── 网页文件/
│   └── 其他文件/
├── 2024年/
│   ├── 文本文件/
│   ├── 网页文件/
│   └── 其他文件/
├── duplicate/              # 重复文件统一存放
└── file_organizer_log.txt  # 日志文件
